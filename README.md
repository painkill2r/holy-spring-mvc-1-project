# 스프링 MVC 상품관리 프로젝트

Spring Boot를 이용한 MVC 기초 학습을 바탕으로 구현한 상품관리 프로젝트

## 학습 환경

1. OS : MacOS
2. JDK : OpenJDK 11.0.5
3. Framework : Spring Boot 2.4.8
    - [Spring Initializer 링크 : https://start.spring.io](https://start.spring.io)
    - 패키징 : jar
    - 의존설정(Dependencies)
        - Spring Web
        - Thymeleaf
        - Lombok
4. Build Tools : Gradle
5. Database : H2
